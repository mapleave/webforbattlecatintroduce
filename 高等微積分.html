最大下界存在證明

S has the least upper bound property B⊂S, B is bounded below,
then α = inf B exists.
B is bounded below => Let L be the set of all lower bounds of B
=> L is bounded above => α = sup L exists
If β < α, β is not an upper bound of L => β ∉ B
If β ∈ B, β ≧ α => α is a lower bound of B => α ∈ L
If β > α, β is a lower bound of B => β ∈ L →←(∵ α = sup L)
=> β is not a lower bound of B => α = inf B exists
